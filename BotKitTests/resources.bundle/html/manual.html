Introduction
An offline notes taking app that stays away from the web and the cloud. It help you create interactive, rich media, rich text documents. It also help you with todo lists, calendar events and alarms, ... etc, while keeping everything offline.
  • It is  ...
    • It takes notes with sketches, photos and audio. .
    • It comes with task specific templates and actions that let you get the job done much quicker.
    • It help you organize your documents with hyperlinks and file manager.
    • It mitigate a lot of security risks and privacy issues by staying completely offline and without any third party content.
  • It is not  ...
    • It is not tiny. 
    • It is not social.
    • It is not the easiest to use.
    • It is not a general purpose web page editor for creating or managing web sites.
